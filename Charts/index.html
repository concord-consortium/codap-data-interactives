Author: Miguel Gutierrez

I made this Data Interactive plugin during the summer of 2017 in order to allow users to make charts in CODAP.
This plugin uses Chart.JS library to add animations and customization capabilities.





Click on a context to open the attribute list.
Click an attribute to begin plotting on the selected chart.



A primary attribute is purple and creates categories along the x-axis.

A secondary attribute is yellow and plots to the y-axis or stacks attributes.
